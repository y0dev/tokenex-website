# TokenEx

This is a landing page to display frontend skills using tailwindcss

## Design

Designed by Ramotion
![Video](/content/ramotion_tokenex.mp4)

## :open_file_folder: Folder structure

- `components` every component involved with the backend
- `dev_data` dummy data for database
- `middleware` all of the middleware components that are used in the project
- `public` frontend src code
- `utils` utilities components used throughout the API
- `views` pug files for the frontend website
